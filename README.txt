
# Jose Lloren - Static Website

This is a static HTML version of Jose Lloren's personal website with an Akatsuki-inspired professional theme.

## Files Included
- index.html - Home page with profile image
- profile.html - Professional profile and experience
- hobbies.html - Hobbies and personal interests
- about.html - Personal background and contact information
- styles.css - All styling for the website
- lovable-uploads/cd43dc96-94c6-40a9-b645-e7e615372386.png - Profile image

## Theme
The website features a professional Akatsuki-inspired theme with:
- Red and black color scheme
- Modern card layouts with semi-transparent backgrounds
- Responsive design
- Subtle animations
- Clean typography

## How to Use
1. Download all files to your computer
2. Keep all files in the same folder structure
3. Open any of the HTML files in a web browser to view the website
4. To publish this website, upload all files to your web hosting server

## Hosting Options
You can host this website on any standard web hosting service like:
- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting (GoDaddy, Bluehost, etc.)

No server-side processing is required as this is a static HTML website.
